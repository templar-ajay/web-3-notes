# web-3-notes
useful notes to revise web03


## install dfx version 0.9.3
```bash
DFX_VERSION=0.9.3 sh -ci "$(curl -fsSL https://sdk.dfinity.org/install.sh)"
```

## create new project
```bash
# project-name = hello
dfx new hello
```
then
```bash
dfx start
```
*split terminal*
```bash
dfx deploy
```
then 
```bash
npm start
```
